# PixelKit Lite

PixelKit Lite is a free, open-source CSS tool collection for designers and developers.  
It includes:

- Color Picker
- Gradient Generator
- Glassmorphism Preview
- Box Shadow Generator
- SVG Editor Lite
- Contrast Checker

## How to Use
Open `index.html` in your browser and start using the tools.
